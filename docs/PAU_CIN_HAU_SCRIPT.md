# Pau Cin Hau Script

A traditional writing system created for the Zomi (Tedim Chin) language
by **Pau Cin Hau**, a Zomi religious leader.

Encoded in Unicode: **U+11D00 – U+11D5F**

## Why It Matters

The Pau Cin Hau script proves that Zomi:
1. Has a **writing tradition designed by a Zomi speaker for Zomi**
2. Has **phonemic tone** — the script marks tones explicitly
3. Has **phonemic vowel length** — built into the vowel system
4. Was recognized by **Unicode** — it's a legitimate world script

Most Zomi speakers today use the Latin alphabet, but this script shows
that the language's tonal nature was understood and encoded long before
the Latin alphabet became standard.
