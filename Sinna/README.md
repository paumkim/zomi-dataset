# Sinna — Zomi Golden Textbooks

> *Sinna* = Learning (in Zomi)

These are original Zolai educational textbooks used when the Zomi language
was actively taught in schools (1980s, Tedim Township, Myanmar). They represent
the **standard reference** for correct Zomi spelling and grammar.

## Why These Matter

These are the books that taught the older generation how to write Zomi
properly. Before them, Zomi spelling was inconsistent. After them, there
was a **standard** that all schools followed. The younger generation who
never had these books in school are the ones drifting from this standard.
